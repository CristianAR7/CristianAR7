## 🧠 Sobre mí

Fundador de **[CRIAL Solutions](https://www.crial.solutions)** — agencia de automatización con IA para negocios locales.

Construyo sistemas que trabajan mientras el dueño del negocio duerme: chatbots web que capturan leads 24/7, agentes de WhatsApp que gestionan reservas y consultas, agentes de voz que atienden llamadas, y landing pages diseñadas para convertir.

- 🔭 Ahora mismo: escalando CRIAL Solutions y su meta-generador de proyectos con IA
- ⚡ Stack favorito: **React + Node.js + Claude API + Twilio**
- 🎯 Obsesión: que la tecnología genere ROI medible, no adornos
- 📬 Abierto a: colaboraciones, proyectos de automatización y clientes con problemas reales que resolver
